# FRC2026

### Important Folders
- /src/main - RIO robot code
- /scripts - python utilities, including vision recording and path mirroring
- /advantagescope-assets - Common assets folder for 3d models/layout

### How to Run: IntelliJ
To the left of the run button, there should be a dropdown. From there,
you can run "Simulate Code", "Deploy Roborio", and "Replay Log File"
